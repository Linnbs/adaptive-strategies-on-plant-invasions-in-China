# adaptive-strategies-on-plant-invasions-in-China
研究背景
植物适应策略（CSR策略）被认为是影响外来植物入侵成功的关键因素，但其与经济用途、居留时间的相互作用如何随空间尺度变化尚不清晰。本研究利用中国栽培外来植物数据集，整合了物种的CSR策略（竞争型C、耐胁迫型S、杂草型R）、经济用途类别数和最小居留时间，旨在揭示这些因素对归化和入侵的直接与间接影响，并探究其在区域尺度上的差异。

分析方法
我们采用贝叶斯结构方程模型，分别在全国尺度和中国七个地理区域尺度上，检验C、S、R得分对归化发生率和已归化植物入侵性的直接与间接效应（通过经济用途和居留时间中介）。同时，结合区域气候（年均温、年降水）和社会经济（人类发展指数）数据，分析路径系数的区域变异驱动因素。
